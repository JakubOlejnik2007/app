# Aplikacja E-Urząd

## O aplikacji
Aplikacja E- Urząd służy do wypełniania wniosków oraz wysyłania ich od
razu do urzędu. Użycie aplikacji pozwala zaoszczędzić wiecle czasu na
szukaniu wzorów wniosków oraz dostarczaniu i do urzędu.
